# Marketing Assets

This folder contains promotional and marketing assets for the XP_Stream mod.

## Modrinth Assets

### Project Icon (`icon.png`)
- **Recommended size:** 512x512 pixels
- **Format:** PNG (with transparency if desired)
- Used as the main project icon on Modrinth

### Gallery Images (`gallery/`)
- **Recommended size:** 1920x1080 or 16:9 aspect ratio
- **Format:** PNG or JPG
- Screenshots and promotional images shown on the Modrinth project page

### Banner (`banner.png`) - Optional
- Promotional banner for featured listings
- Check Modrinth's current guidelines for recommended dimensions

## Notes
- These assets are NOT included in the mod JAR file
- They are uploaded separately via the Modrinth web interface
- Keep source files (PSD, XCF, etc.) in a `source/` subfolder if needed
